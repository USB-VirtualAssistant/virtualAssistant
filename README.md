# Virtual Assistant

## Requirements

- Python 3.0+

## Installations

### ASR Whisper Client

```bash
$ sudo apt update && sudo apt upgrade
$ pip install fastapi
$ sudo apt install uvicorn
$ pip install python-multipart
"Run service"
$ cd src/main/java/org/fundacionjala/virtualassistant/service
$ uvicorn ASRClient:app --reload
```
